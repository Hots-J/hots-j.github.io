LeetCode 51. N皇后

发表于 2020-05-16 分类于 LeetCode

1. 原题链接

51. N 皇后

2. 题目描述

n 皇后问题研究的是如何将 n 个皇后放置在 n×n 的棋盘上，并且使皇后彼此之间不能相互攻击。

上图为 8 皇后问题的一种解法。

给定一个整数 n，返回所有不同的 n 皇后问题的解决方案。

每一种解法包含一个明确的 n 皇后问题的棋子放置方案，该方案中 'Q' 和 '.' 分别代表了皇后和空位。

示例：

提示：

皇后彼此不能相互攻击，也就是说：任何两个皇后都不能处于同一条横行、纵行或斜线上。

3. 思路

对于一个二维数组，我们可以通过i+j来表示从左下角到右上角的对角线，通过i-j+n-1来表示从左上角到右下角的对角线，i，j为数组下标，从0开始

判断每一行当前位置是否可以放一个皇后，若满足条件，在放置皇后，进入下一行，若当前行的所有位置都无法放置皇后，则回溯到上一行，删除上一行皇后的位置，并找下一个合适的位置。直到最后放满所有行
